## 使用  
页脚的图片顺序是按动画出场顺序  
  
添加不蒜子访问量计数器  
不需要的话可以在`rabbit.js`修改   
`var busuanzi = false;`  
然后重新生成min文件 
* pv 每访问一次+1
* uv 每一个用户只+1
